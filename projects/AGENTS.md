# projects/AGENTS.md

本目录是 PPT 生成工程的工作目录，不是研究材料目录。进入本目录前先读根目录 `AGENTS.md`。

## 作用

- 存放开题答辩 PPT 的自动生成工程文件：模板分析、内容填充、版式修复、导出和验证。
- 这是工具链工作区，不是研究产出的正式存储位置。

## 内容说明

`opening_defense_20260712/` 是 2026-07-12 开题答辩 PPT 的生成项目：

- `analysis/`：学校 PPT 模板结构解析、填充计划生成。
- `validation/`：生成后 PPT 的版式检查、边界审计、内容读回验证。
- `exports/`：生成的 PPTX 导出文件。

## 边界

- 不存放正式实验数据、结果报告、学习材料或项目规则。
- PPT 的最终正式版本放在 `opening/slides/`，不在这里。
- 图资产放在 `figures/`，不在这里复制。
- 本目录的工具脚本和中间文件不作为项目文档入口。

## 规则

- 修改 PPT 生成逻辑后，必须跑 `validation/` 中的检查脚本验证。
- PPT 内容变更必须基于 `opening/report/opening_report.md` 和 `figures/` 中的正式图。
- 生成过程中的旧版 PPTX 可以删除，保留最新可用版本即可。
