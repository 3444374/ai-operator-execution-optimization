# notes/

本目录维护和导师、企业侧沟通相关材料。进入本目录前先读根目录 `AGENTS.md` 和本目录 `AGENTS.md`。

## 文件

| 文件 | 内容 | 什么时候读 |
|---|---|---|
| `AGENTS.md` | 本目录规则 | 整理沟通材料前读 |
| `communication_notes.md` | 待确认问题、沟通话术、会议结论 | 准备导师/企业沟通 |

## 与其他目录的关系

- 项目方向和规则：根 `AGENTS.md`、`PROJECT_OUTLINE.md`
- 调研材料：`research/`
- 实验结果：`motivation/results/`、`feasibility/results/`
- 开题材料：`opening/`

## 更新规则

- 企业侧或导师反馈改变课题边界时，同步检查根 `AGENTS.md`、`PROJECT_OUTLINE.md`、`overview/` 和 `opening/report/`。
- 已解决、过期或被新结论替代的问题要移出当前待办。
