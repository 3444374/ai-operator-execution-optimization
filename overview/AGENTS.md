# overview/AGENTS.md

本目录维护项目总纲和当前路线。进入本目录前先读根目录 `AGENTS.md`。

## 作用

- 记录课题定位、用户目标、阶段计划和方向边界。
- 不放实验原始结果，不放长篇文献摘录，不放代码实现细节。

## 更新规则

- 当课题方向、业务场景、实验结论发生变化时，同步更新 `current_direction_and_plan.md`。
- `project_outline.md` 保持稳定，只记录高层定位和目录结构。
- 每个判断要区分：已验证事实、本地实验结论、合理推断、待确认问题。

## 当前边界

- 主线应服务 AI infra / inference infra 能力积累。
- 数据库场景是落地入口，不把项目带回传统数据库内核 GPU 查询算子。
- 当前推荐场景是数据库内置 AI 算子背后的批量 embedding / RAG 数据准备链路。
