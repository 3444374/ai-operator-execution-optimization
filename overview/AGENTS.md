# overview/AGENTS.md

本目录维护项目总纲的快速参考入口。进入本目录前先读根目录 `AGENTS.md`。

## 作用

- `current_direction_and_plan.md`：当前方向的快速参考卡片（TL;DR），提供 2 分钟可读完的课题概览：定位、技术栈、研究内容、优先级、关键证据和完整文档入口。

## 边界

- 不放完整研究内容定义和实验路线（以根 `PROJECT_OUTLINE.md` 为准）。
- 不放文献依据和知识库（以 `research/knowledge_hub.md` 为准）。
- 不放原始 CSV、实验结果（以 `motivation/results/` 和 `experiments/results/` 为准）。
- 不替代任何根目录主文档；本文档只是快速入口，冲突时以根目录文档为准。

## 规则

当以下内容变化时，同步更新 `current_direction_and_plan.md`：

- 课题方向或技术栈变化；
- 研究内容调整；
- 主场景变化；
- 实验优先级重排；
- 关键证据变化。

- 保持在一页以内（约 100 行），不膨胀为第二个 PROJECT_OUTLINE.md。
- 每个判断要区分事实和待验证。
