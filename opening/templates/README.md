## 模板使用原则

- Word 模板用于约束报告结构和格式，不直接改变研究口径。
- PPTX 模板用于约束版式、安全内容区、字体、页眉页脚和母版，不当作背景截图使用。
- CCF 目录用于优先筛选权威文献，尤其是 CCF-A 会议和期刊。
